# portal-cn-lab

一个用于归档和发布多个独立 HTML 页面的实验室仓库。

## 项目简介

本仓库旨在收集、整理并托管一系列独立的 HTML 页面。这些页面可能包含实用工具、信息展示、原型演示或其他静态内容。仓库本身不绑定特定域名或网站，仅作为页面资源的集中存放与版本管理空间。

## 目录说明

```
.
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 页面引用的公共资源（如 CSS、JS、图片等）
├── docs/           # 相关文档或说明
└── README.md       # 本文件
```

- **pages/**：每个子文件夹或文件对应一个独立页面，内部可包含完整的 HTML、CSS 和 JavaScript。
- **assets/**：存放多个页面可能共享的静态资源，避免重复。
- **docs/**：用于存放页面使用说明、设计文档或其他参考资料。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开运行。
- 页面之间相互独立，无依赖关系（除非明确引用公共资源）。
- 归档的页面可能处于不同完成阶段：实验性原型、功能完整版、或仅供存档的旧版本。
- 每个页面文件夹内建议附带简短说明（如 `README.md`），描述其用途、作者及注意事项。

## 维护说明

- 欢迎提交 Pull Request 添加新页面或改进现有页面。
- 添加新页面时，请在 `pages/` 下新建文件夹，并确保页面完整可运行。
- 请勿在页面中包含任何具体域名、网址、营销承诺或诱导点击内容。
- 如有资源更新或页面调整，请保持目录结构清晰，避免文件冲突。
- 本仓库由社区共同维护，请遵守基本的开源协作规范。

## 许可

本仓库内容默认采用 MIT 许可证，具体页面可能包含单独的许可声明。请在使用前查阅相关文件。
